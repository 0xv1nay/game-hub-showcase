# Arcadia - Retro Arcade Game Collection

A static HTML/CSS/JS website for hosting retro arcade games.

## Structure

- `index.html` - Main page
- `script.js` - Game filtering and search logic
- `public/` - Static assets

## Deployment

This site is configured for GitHub Pages. Push to the `main` branch to deploy automatically.

## Adding Games

Edit the `games` array in `script.js` to add or remove games.
